## 🚀 Cómo usarlo

**Doble clic en `index.html`.** Ya está. Sin servidor, sin `npm install`, sin nada.

## ✨ Funcionalidades

| | |
|---|---|
| 📋 **Listar** | Muestra el catálogo en una tabla |
| ➕ **Añadir** | Formulario para meter juegos nuevos |
| 🗑️ **Borrar** | Un botón por fila |
| 🔍 **Buscar** | Filtra por título o género al escribir |
| 🔢 **Contador** | Cuántos juegos hay ahora mismo |
| 📥 **Importar** | Carga tu propio `datos.json` |
| 📤 **Exportar** | Descarga el catálogo como JSON |

## 📁 Archivos

```
clase-3/
├── index.html    → la interfaz
├── app.js        → la lógica
├── datos.js      → catálogo inicial (se carga al abrir)
└── datos.json    → el mismo catálogo, para importar/exportar
```

## 🌿 Ramas

```
main  ←  pre  ←  dev
```

| Rama | Para qué |
|---|---|
| `main` | 🟢 Versión estable |
| `pre` | 🟡 Pruebas antes de publicar |
| `dev` | 🔵 Desarrollo del día a día |

> **Regla del proyecto:** un commit por funcionalidad, siempre empezando en `dev`.
